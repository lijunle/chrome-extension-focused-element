# Privacy Policy

**Last updated: November 26, 2025**

## Overview

Focused Element is a Chrome extension that helps developers identify the currently focused element on a webpage.

## Data Collection

**This extension does not collect, store, or transmit any user data.**

Specifically, this extension:

- Does not collect any personal information
- Does not track browsing history
- Does not store any data locally or remotely
- Does not transmit any data to external servers
- Does not use cookies or any tracking mechanisms
- Does not access or modify any user files

## Permissions

This extension only uses the minimum permissions required to function:

- **activeTab**: Used solely to identify and highlight the focused element on the current page when activated
- **scripting**: Used to inject the script that detects and highlights the focused element on the page

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last updated" date at the top of this document.
